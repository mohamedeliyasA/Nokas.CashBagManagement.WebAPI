# Nokas.CashBagManagement.WebAPI

•	Nokas Cash Handling specializes in the secure management of physical cash for businesses, banks, and public institutions. Clients benefit from safe cash transport, accurate counting, and direct bank deposits.
•	The public-facing cloud API for Nokas Cash Handling will facilitate smooth communication between vendors and Nokas.
•	This API will enable vendors to seamlessly interact with Nokas' services for Cash Bag Registration and Cash Bag Counting Result, enhancing transparency and speeding up cash handling operations.
